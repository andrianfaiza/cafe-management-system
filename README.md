# Cafe Management System

A simple Laravel-based cafe management system.  
This project helps manage cafe operations such as tables, menu, customers, and transactions.

## 🚀 Features
- Manage tables
- Menu management
- Customer management
- Transaction system
- Dashboard

## 🛠️ Tech Stack
- Laravel 12.55.1
- Blade Templates
- MySQL
- JavaScript (for dynamic forms)

## 📖 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/andrianfaiza/cafe-management-system.git

2. Install dependencies:
    ```bash
    composer install
    npm install && npm run dev

3. Configure .env file (database, app key)
4. Run migrations:
    ```bash
    php artisan migrate
5. Start the server:
   ```bash
   php artisan serve
